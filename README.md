# Patient Health Database


Author: Pranali Rane

## Purpose
The code in this repository will run a patient health database.

The code is written in Python.

## Specifications
The Database will contain:
* Patient Name
   + First Name
   + Last name 
* Age 
* address
* Blood type
* Test Results
   1. HDL
      + High density Lipoprotein
   1. LDL
      + Low density Lipoprotein
